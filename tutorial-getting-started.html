‚ùì Frequently Asked Questions

Is my data really private?

Absolutely. Every tool processes data entirely within your browser using client-side JavaScript. No information is transmitted to our servers or any third party. You can verify this by checking your browser's network activity while using the tools.

Do I need to create an account?

No! We deliberately avoid user accounts to maximize privacy. You can use any tool immediately without signing up, logging in, or providing any personal information.

Will my work be saved if I close the browser?

Most tools (Scratchpad, Project Planner, To-Do List) use local storage to persist data between sessions. However, the PDF Editor operates on files you provide and doesn't store them permanently. Always download your merged PDFs before closing the page.

Can I use these tools offline?

Once a tool page is loaded, most functionality works offline. The PDF Editor, Scratchpad, Project Planner, and To-Do List all operate without requiring an internet connection after the initial page load.

Are there any file size limits?

The only limits are those imposed by your browser and device hardware. The PDF Editor can handle multiple large files, but processing very large documents may take longer and consume more memory.
