üîß Core Features Explained

üìÇ File Selection and Upload

The PDF Editor supports two methods for adding files:

  • Drag and Drop: Simply drag PDF files from your file explorer directly into the designated drop zone. The editor highlights the area when files are detected.
  • File Browser: Click "Select PDF Files" to open your system's file browser and choose one or multiple files at once. Hold Ctrl (Windows/Linux) or Cmd (Mac) to select multiple files.

Pro tip: You can add files in multiple batches. If you forget a file, simply add more files and they'll be appended to your current list.

üîÑ Reordering Documents

The order in which files appear in the list determines the order they'll be merged. To reorder:

  1. Look for the ‚ò∞ (hamburger) icon next to each file name
  2. Click and hold on this icon
  3. Drag the file to its new position
  4. Release to drop it in place

This is crucial when creating documents with a specific flow, such as table of contents followed by chapters in order.

üóëÔ∏è Removing Files

Made a mistake or added the wrong file? Each file in the list has a remove button (usually an X or trash icon). Click it to remove that specific file from the merge list without affecting others.

üìù Metadata Customization

Before merging, you can set professional metadata for your combined PDF:

  • Title: The document's official title (appears in PDF readers and file properties)
  • Author: Your name or your organization's name
  • Subject: A brief description of the document's content
  • Keywords: Comma-separated terms for searchability and organization

While optional, metadata makes documents more professional and easier to organize in large collections.

‚ú® Advanced Techniques

Creating Document Portfolios

Combine multiple related documents into a single portfolio PDF:

  1. Gather all documents you want to include (resumes, certificates, letters of recommendation)
  2. Add them to the PDF Editor in logical order
  3. Set metadata with your name as author and "Professional Portfolio" as title
  4. Merge and download

Combining Research Papers

Academic researchers often need to combine papers for review or submission:

  1. Add all papers in chronological or thematic order
  2. Use metadata to indicate the collection purpose (e.g., "Literature Review - Machine Learning 2024")
  3. Add relevant keywords for easy searching later

Creating Multi-Chapter Documents

Authors and content creators can assemble chapters into complete books:

  1. Ensure each chapter is a separate PDF file
  2. Add files in chapter order (Chapter 1, Chapter 2, etc.)
  3. Set comprehensive metadata including book title and author
  4. The resulting PDF maintains the page numbers and formatting of each chapter

üí° Professional Tips

üìå Tip #1: Verify Before Merging

Always double-check the file list before clicking merge. Ensure files are in the correct order and you haven't accidentally included duplicates. This saves time and prevents errors.

üìå Tip #2: Use Descriptive Filenames

The downloaded file is automatically named based on your first file. Use descriptive names for source files (e.g., "Annual_Report_2024_Part1.pdf") so the merged file gets a meaningful name like "Annual_Report_2024_Part1-merged.pdf".

üìå Tip #3: Process Large Files in Batches

If working with many large files, consider merging in batches of 5-10 files, then combining the results. This prevents browser memory issues and provides better control over the final structure.

üìå Tip #4: Leverage Metadata for Organization

When creating document collections, use consistent metadata conventions. For example, all project proposals could use "Proposal" as a keyword, making them easy to find later.

üìå Tip #5: Preview After Merging

The merged PDF displays in the browser before download. Always scroll through to verify everything merged correctly, pages are in order, and content is readable.

‚ö†Ô∏è Password-Protected PDFs

The editor attempts to handle encrypted PDFs with the ignoreEncryption option, but heavily protected files may fail to merge. If possible, remove passwords before merging.

‚ö†Ô∏è Browser Memory Limitations

Very large PDFs (100+ MB each) or merging 20+ files simultaneously may cause browser slowdowns or crashes. If this happens, try working with smaller batches or close other browser tabs to free up memory.

‚ö†Ô∏è Non-PDF Files

The editor only works with PDF files. Attempting to add images, Word documents, or other formats will result in errors. Convert files to PDF format first using your operating system's built-in tools.
