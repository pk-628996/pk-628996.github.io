Use Split-View Workflows

Open multiple tools side-by-side (e.g., Scratchpad + To-Do List + PDF Editor) to create a focused workspace. On macOS use Split View; on Windows use Win + Arrow keys.

Set Daily Intentions

Start your day by listing 3 MITs (Most Important Tasks) in the To-Do List, then plan milestones in the Project Planner.

Template Your Work

Maintain starter notes in Scratchpad (e.g., meeting notes, bug report template). Duplicate and reuse to save time.

Time Blocking

Schedule focused blocks for deep work. Use the Planner to define start/end times and track progress.

Clear the To-Do List daily. Archive done tasks and move pending items to the next day intentionally.

Large PDF Merges Stall

Close other tabs to free memory, merge in batches, or use smaller files.

Check if your browser cleared site data. Restore from your periodic exports or backups.

Mobile UI Feels Cramped

Rotate to landscape, zoom out to 90%, or switch to a tablet/desktop for heavy tasks.
